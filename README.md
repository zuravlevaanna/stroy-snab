# СтройСнаб

Система управления поставками стройматериалов

## Структура репозитория
- **MobileApp** — приложение для кладовщика (приёмка/отгрузка)
- **WebApp** — панель менеджера по закупкам
- **ServerApi** — серверная часть учёта товаров
- **Docs** — документация
