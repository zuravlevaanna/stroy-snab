# Требования к СтройСнаб

## Функциональные требования
- Учёт товаров на складе (приход, расход, остатки)
- Управление поставщиками
- Создание заказов на поставку
- Приёмка товара по штрихкоду
- Отчётность по движению товаров
- Оповещение о низких остатках

## Нефункциональные требования
- Быстрый поиск товара по штрихкоду
- Работа в офлайн-режиме (для кладовщика)
- Простой интерфейс для работы со сканером
